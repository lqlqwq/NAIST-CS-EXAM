#### 1.语言学习

笔者一开始花了很多时间学习日语，甚至在来日本之前想着自己应该如何在语言学校好好学习日语。然而日语成绩在笔者的修考过程中并没有起到非常明显的作用。此外，很多学校仅以笔试成绩的高低决定合格与否，面试成绩几乎不影响。

但是不可否认日语的水平(尤其是口语)对于日本的生活以及之后找工作是非常有帮助的，请自行平衡在日语学习方面投入的时间。



英语成绩在几乎所有学校的修考分数中都占一定比重，请尽量保证至少达到你想去的学校的平均分。

在这笔者给一个粗略的参考线：对于所有只收托福的专攻，至少达到80,有90最好。 对于上位学校如帝国大学和较好的国公立，建议托业不低于750分。对于MARCH以及一般的国公立，建议托业不低于650分。



此外，笔者推荐一个非常好的练习口语的[网站](https://www.italki.com/)。通过这个网站可以方便的一对一练习口语，笔者在学习日语期间找了个日本人大概是70一小时。



#### 2.来日途径

众所周知考学途径主要分为三种，语言学校，研究生，直考。笔者仅代表自己简单分析一下三者的优劣势。



语言学校(最长两年)：[(语言学校评价表)](https://docs.qq.com/sheet/DRmJVUGpWVFZ3YkNI)

笔者自己是通过语言学校这一途径来日本的，但是在考学的过程中其实非常后悔。因为语言学校每天都要上半天课，再加上自己本身的惰性，一天的学习时间并不多。语校上午班还可能会导致下午没有干劲(想睡觉)。语言学校的大部分老师其实不错，上课干些别的，比如做题背单词之类的，基本上不会管。但是偶尔可能会遇到非常麻烦的老师，上课要求很严，这种老师会非常难受。

此外，请尽量找一个 **考试请假不算缺勤** 的语校。不然夏季考的时候，如果一个月多报几个学校，出勤率会非常难看。

当然，语校也是有优点的。个人觉得语校最重要的一个优点，是能在日本认识很多同样是考大学院的人(并不一定只是在语校，比如私塾，语校提供了来日本的机会)或已经考完的前辈，大家可以互相交流分享经验，互相共勉。

此外，如果想练习日语的话打工也是个不错的选择，也可以学到一些社会文化之类的，对日后想在日本工作的人会有帮助。但是打工会进一步压缩学习时间，所以个人并不是很推荐打工。



研究生(一般为一年)：

研究生作为来日本的另一个途径，有一定的门槛，优点是可以提前进研究室，缺点是需要先陶瓷。陶瓷就要求投入精力，需要写陶瓷信，练习口语，准备面试等一系列前置工作，相比于几乎没有门槛的语言学校，要花上不少的时间。

研究生大体分为两类：强内诺制的与弱内诺制的。

强内诺制代表，当你成功陶瓷上研究生之后，以研究生身份参加本学校修士考试，成功概率非常高。这里笔者列举两个比较典型的强内诺制学校：东北大学，北海道大学。

弱内诺制自然相反，研究生身份几乎对于本学校的修士考试没有帮助，仅仅是为了拿到签证。请小心部分会给学生分配很多任务的弱内诺研究室，比较方便的办法就是在申请之前给研究室的中国人前辈(如果有的话)发一封邮件询问一下。

笔者个人觉得研究生比较适合背景比较好的或者有明确的研究方向的朋友。



直考：

没啥好说的国内复习完直接来考试，优点是省钱，缺点是受限于签证时间，可能并不能参加很多学校的考试。此外，部分学校要求有在留卡才能出愿，但是这种学校非常少，直接跳过即可。



#### 3.私塾选择

私塾并不是一个必选项，尤其是对于基础较为扎实的朋友。此外，私塾存在非常多的虚假宣传，招生老师会想尽一切办法把学生忽悠进来，在确定报名之前非常推荐从多方面了解私塾的情况，比如去线下试听，并询问一些已经在该私塾上课的人。

由于绝大部分私塾的老师/班主任，实际上是在读硕士/博士兼职，因此指望私塾的老师/班主任非常的认真负责几乎是不可能的事(有但是很少)，不要幻想老师/班主任会督促你学习，一切全靠自己主动。



私塾可能能提供的帮助：交友，数学课和专业课，学校选择，研究计划，模拟面试，过去问等。

总结：私塾本身，或者说私塾整体并不可靠，但是私塾内有可靠的人(老师/同学)，笔者觉得私塾最重要的意义是去认识这些可靠的人。



#### 4.学校选择

学校选择是非常主观的部分，根据每个人的情况会有很大的不同，笔者这边仅讲述一下自己的思路以及一些可能会影响到抉择的点。



首先，请明确自己是否有感兴趣的方向，还是打算随便找个研究室读两年拿个修士文凭。笔者个人认为，在日本考学是一个非常好的转专业/追求自己感兴趣的方向的机会，因为日本的许多学校/研究室并不太在意学生本科的专业。



其次，需要考虑的是修士毕业后的进路，是打算在日本找工作还是读博。

笔者觉得如果是打算修士毕业后在日本找工作的话，那么具体的分野和研究室的强弱就没有那么重要，学校的排名和位置会更加重要一点，比如首都圈的学校找工作的时候相对有优势。此外，很多学校/专攻都有就职率这一数据，在寻找目标学校/专攻的时候也可以关注一下。

如果是打算读博的话，在日本读博会相对简单一些。研究室内进学(留在修士的研究室继续读博士)基本上是非常轻松的。如果打算博士去别的国家的话，研究室的产出，Connection 等就会比较重要。



然后，日本的大学院入学考试基本分为两种类型，笔试型和面试型(包括强内诺学校)，笔者觉得二者并不可兼得，尽量全选择笔试型/面试型会比较好。部分学校虽然也有笔试，但是倍率几乎是1:1，实际上是面试型/强内诺学校，比如东京都立大学的冬入。



根据笔者的思路，笔者分析一下自己的择校过程

笔者本科是机械自动化但是完全不感兴趣，当时仅仅是觉得就业还不错才选择的，毕业后也没有找相关的工作而是直接选择了转码。笔者在刚开始考学时也没有想好某个特定的方向，而是在寻找学校的过程中，看到了有在做AR/VR/HCI相关研究的研究室，感觉还挺二刺猿的，加上笔者之前也或多或少接触过一些相关的东西，因此顺着这条路开始寻找具体的研究室。

笔者虽然是个二刺猿但是并不想留在日本，打算去其他国家读博。笔者在对NAIST的研究室感觉感兴趣之后，首先发了封邮件给研究室的国人前辈进行了交流，得知大老板在这个领域有一定名气(H-Index 50+)，研究室的产出和氛围也很不错。接着笔者就去研究室进行了为期三天的见学，在见学过程中笔者和坐在旁边的助教(之后变成了小老板)用英语交流了很多研究相关的东西，助教也对笔者的RP给予了很多指点。同时笔者也对研究室进行了调查，发现研究室有许多国际上的交流，比如交换生/访问之类的，之前在研究室工作过的两位小老板现在也回到自己的国家建立了自己的研究室。因此，笔者在权衡了自己的背景，水平后，觉得NAIST的研究室大概是自己能力范围内最优的选择。

但是只有最想去的学校还不够，不能把所有的鸡蛋放在一个篮子。笔者觉得自己在研究计划和面试的准备上已经花费了很多时间，这些经验对于别的学校的陶瓷/考试仍然有用，因此后续的择校仍然是以面试型学校为主，如JAIST，东京都立，筑波，东北等。



题外话：笔者在陶瓷经历里提到了由于想去NAIST拒绝了一位东京都立的教授，这位教授非常的亚撒西，在得知笔者由于时间冲突不想放弃NAIST的考试机会只能拒绝都立后(NAIST三月才考试，都立一月考试，合格后我必须在二月底前答复)，不仅没有生气，反而对笔者进行鼓励，同时表示要是笔者23冬入落了NAIST那么24夏还是可以考他的研究室，参观完研究室之后还把笔者亲自送到学校门口，笔者自己感到非常愧疚，因此希望帮这位教授宣传一下，这是他的[研究室链接](https://hci.fpark.tmu.ac.jp/research.html)，目前主要在进行触觉相关的研究。



#### 5.陶瓷相关

[笔者参考过的陶瓷信模版](https://zhuanlan.zhihu.com/p/376491584)

首先是如何写陶瓷信，就笔者个人而言，陶瓷信主要分为3部分结构。第一部分简单介绍一下自己并且提出要申请硕士，第二部分简单提一下自己的附件，比如简历和个人陈述，RP等，第三部分结合自己感兴趣的东西和目前研究室的方向，写一些自己的想法，尽量表达出对于该研究室有过认真的了解。

其次，部分教授的邮件地址可能并不好找。最简单的办法就是查教授发表过的文章进行下载，一般文章里都会附上联系方式。

此外，部分研究室会提供学生的邮件地址，可以尝试联系研究室里的中国人前辈，有一些前辈比较热情，会愿意帮忙修改陶瓷信/RP。

笔者不太推荐的是过长的陶瓷信以及海投，请尽量每一封邮件都根据研究室写一点独特的东西。



#### 6.出愿相关

募集要项相关：

请务必仔细阅读感兴趣的学校的募集要项。比如部分学校要求外国人考生在出愿前一段时间提前登录，如果错过了就无法出愿。



出愿材料相关：

请务必准备纸质的成绩单，毕业证明，这两份材料是所有学校都需要的。此外，日本的部分学校不承认电子章，即学校打印机打印的文件，可以的话最好准备盖上鲜章的材料。有一些学校还会要求提供学信网相关的证明。

针对在国内进行考试的托福考生，非常建议再考一个托业。因为部分学校在出愿时要求托福原件，有些甚至在出愿完成后不归还原件(~~说的就是东工~~)，然而国内托福无法增送原件。







## NAIST(奈良先端科学技术大学院大学)2024合格体验记

由于笔者主要的目标是NAIST，因此仅简略的介绍一下JAIST的考试。

### JAIST(北陆先端科学技术大学院大学)

JAIST的计算机分为两个部分，知识科学和情报科学，不管从哪个入口合格，入学后可以选择的研究室都是互通的，据说知识科学会比情报科学容易合格。

JAIST的大部分研究室不需要陶瓷，且面试时考官比较随机。面试主要是7分钟的PPT发表加上23分钟的问答(日/英均可)，共计30分钟。笔者参加了两次面试，均全部围绕研究计划书进行问答，据说也有部分人会被问到一些简单的数学问题。

2025年更新：猜测由于老中鸽子太多，因此JAIST收紧了考试政策，没有在留卡仅能海外特别出愿，变相强制要求内诺，有在留卡的政策暂时未变。



### NAIST(奈良先端科学技术大学院大学)

#### 一.内诺/出愿

NAIST目前的考试制度有点类似于强内诺，对于外国人来说必须要获得教授的许可才能出愿，且**外国人必须在出愿两个月前进行预申请**，因此需要提前较长的时间准备。虽然NAIST出愿时需要填五个志愿，但是~~外国人仅会被分配到第一志愿~~（已失效，详见4.24更新）。

但是NAIST的内诺并不难拿，因为NAIST有学校官方举办的春季/夏季Seminar(类似于见学会)。此外，还有学校官方的见学活动(线上/线下均可，最长三天)，由于通过学校申请因此个人猜测研究室必须回复。

由于NAIST的考试中研究计划占了相当大的比重，笔者非常推荐在陶瓷/见学之前，就先写好初版的研究计划，这样在见学的时候就可以直接跟教授/助教/学生聊，对于后续研究计划的修改很有帮助。

笔者的流程：笔者首先写好了陶瓷信和研究计划，然后通过研究室的主页联系上了一位研究室内在读的硕士，在他的帮助下修改了陶瓷信和研究计划。随后笔者给大老板和小老板一起发了邮件但是未回，遂通过学校的见学活动申请了线下见学三天，当天就收到了小老板的回复。



**2025.4.24更新**：笔者在入学后遇到了一位一开始没有被分进第一志愿调剂到别的志愿的朋友，后续可能他第一志愿的研究室有人辞退，因此他又被重新分到了第一志愿。因此NAIST并非没有调剂，请认真填写第二到第五志愿，但是确实非常罕见，笔者目前仅知道这一例情况。



#### 二.考试

写在最前：如果你对NAIST有兴趣，**请务必参加夏季考**，每年三次考试的难度是：夏入(7月)>>秋入(10月)≥冬入(3月)。具体的分析请见下文。



NAIST的考试的分数结构分为四部分，满分200分：书类审查(50分)，英语(30分)，数学(30分)，面试(90分)。

书类审查：没啥好说的，这是大部分人无法改变的分数，主要是根据本科GPA折算的成绩。对于GPA较差的学生不用太过担心，应该有类似于保底的分数，笔者的GPA就相当难看，但是推测的书类审查分数肯定没有太低。

英语：建议刷个托业，笔者托福80出头但是感觉折算的分数并不会太高因此在考前又考了个托业。此外，NAIST支持多邻国。同时NAIST可以在出愿后到考试前的一段时间增送英语成绩，具体请看官网。NAIST英语入试的平均分相当恐怖，因为有不少英语母语的人，笔者记得学校官方曾给过数据，在2023年英语入试的学生中托业平均分达到了900+。不过也不用太担心英语成绩，英语只占很小的比重并且也拉不开差距，笔者推荐刷一个800+的托业成绩应该够用了。

数学：NAIST的数学考试是比较简单的微积分和线性代数，并且给出了参考书，题目和参考书的课后习题一模一样，笔者建议有空至少把参考书粗略的看一遍(尤其是线性代数)，不一定要做课后习题，因为课后习题太多了。此外NAIST的数学考试形式较为特别，是在面试时当着考官的面现场做题，需要 写出做题过程+向考官解释如何解题，在遇到不会做的题目卡住时甚至可以向考官寻求帮助([NAIST数学考试流程](https://isw3.naist.jp/Admission/img/nyushi/online-math-exam-guideline_jp.mp4))。总而言之，NAIST的数学考试的题目较为简单且有固定的题目池，笔者基于自己遇到过的和朋友提供的题目整理了一份过去问。[数学过去问(更新时间2025.4.24)](https://1drv.ms/w/c/d5d205f867607827/ES3Wtk3IKZREmKWZv_Zh3JkB4haBaNeAGowaY7VU8F3-WA?e=bvsD03)

面试：面试占90分，几乎决定了考生能否合格。面试共持续12分钟左右，有三位教授，第一研究室的大老板/小老板一定会在且作为主面试官(个人推测剩余两位教授必定是第二到第五志愿研究室的教授)。一般会先让考生简短的介绍自己的研究计划，在考生介绍研究计划的同时教授会浏览考生所提交的研究计划，然后围绕研究计划进行问答。因此，笔者推荐研究计划最好要有流程图之类的可视化部分。

整体的考试流程：提前半小时进入房间等待-数学考试(10分钟)-稍微等待准备口试-口试(12分钟)-结束

## 日本留学生活指北

### 一.来日前准备

#### 1.支付

国内银行卡部分(日元取现，信用卡消费)，可以参考[这个帖子](https://zhuanlan.zhihu.com/p/649834833)



#### 2.租房

提到租房，就不得不先补充一些日本的铁道知识。

日本电车有许多线路存在 各站停车(每站都停) 和 快速/特快/急行(跨站车)，所以有些地方虽然住得远但是可能通勤反而更快。

拿与语言学校比较多的新宿/高田马场接轨的西武新宿线举个例子：

左边的不同颜色代表不同种类的列车，灰色是最慢的各站停车，旁边的青色则是准急。以住在第八站都立家政和第九站鹭之宫的人进行对比，虽然住在第九站鹭之宫的人离底站西武新宿更远，但是由于鹭之宫可以乘坐准急列车，因此到达西武新宿所花的时间反而小于住在第八站的只能乘坐各站停车的人。

关于具体的租房地址，建议视自己语言学校/大学院的位置而定，此外都内有一条最重要的环线：山手线。如果住在山手线沿线那么出行会变得非常方便。但是相应的房价也不便宜。使用谷歌地图可以比较准确的查看通勤时间以及价格.



碎碎念：

对于没有入境(没有拿到在留卡)的人来说，如果想提前租房，那么只能租可以海外审查的房子，相应的房源数量会变少。此外，如果在留学生入境的高峰期，比如每年的四月左右，房源会相当的紧张。因此有许多人都选择先住一阵子学校的宿舍/寮，过几个月再自己搬家。不过由于日本这边高昂的人力成本以及大部分房子都不带家具，搬家实际上比较麻烦。笔者是跟一位在日本上班的朋友合租的，因此租房的时候也不用海外审查，甚至每个月租金也可以直接支付宝转给朋友，非常方便。
